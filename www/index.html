Welcome to AncuiJS

A modern JavaScript visualization library that makes it easy to present rich, data-driven visuals.

A fresh Declarative approach to visualizations

Use an expressive list of functions to describe the visual with little to not pre-setup.
const barsData = $data([50, 20, 30, 10, 50]);
const ctx = container("bar-1", $id("example-bar-1"));

renderVisuals(ctx,[
    bars(barsData, [$maxHeight(50), $yOffset(30), $width(150)])
    ]);

Who does it work?

The maximum height of each bar is controlled by $maxHeight(...), while the space between the bars is determined by $yOffset(...).
